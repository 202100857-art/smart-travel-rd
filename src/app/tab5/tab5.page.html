Acerca de

Smart Travel RD

Versión 1.0.0

Smart Travel RD es una aplicación móvil desarrollada como proyecto académico para la asignatura Programación de Dispositivos Móviles.

Funciones principales

  • Exploración de destinos turísticos.
  • Mapa y geolocalización.
  • Cámara y lector de códigos QR.
  • Audioguías en línea.
  • Favoritos y modo visual configurable.
  • Autenticación mediante Firebase.
Desarrollado por Luis Alfonso Marte Del Villar Universidad Abierta para Adultos — 2026

Política de privacidad

Resumen para la demostración

Smart Travel RD utiliza la ubicación, cámara y conexión a Internet únicamente cuando una función lo requiere y el usuario concede el permiso.

La autenticación se gestiona mediante Firebase Authentication. La aplicación no almacena contraseñas directamente.

Esta versión tiene fines académicos y no comercializa ni comparte información personal con terceros.
